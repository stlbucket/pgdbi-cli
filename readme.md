
# pgdbi-cli

> My awesome command-line tool


## Install

```
$ npm install --global pgdbi-cli
```


## Usage

```
$ pgdbi --help

  Usage
    $ pgdbi [input]

  Options
    --postfix  Lorem ipsum  [Default: rainbows]

  Examples
    $ cli-name
    unicorns & rainbows
    $ cli-name ponies
    ponies & rainbows
```
